## 📦 Instalación Local

1. Clona el repo: `git clone https://github.com/leoch17/portfolio-web.git`
2. Instala dependencias: `npm install`
3. Configura las variables de entorno en un archivo `.env.local`
4. Inicia el modo desarrollo: `npm run dev`
5. Para exportar el sitio estático: `npm run build` (asegúrate de tener `output: 'export'` en tu `next.config.js`)

## 📦 Local Installation

1. Clone the repo: `git clone https://github.com/leoch17/portfolio-web.git`
2. Install dependencies: `npm install`
3. Configure environment variables in a `.env.local` file
4. Start development mode: `npm run dev`
5. To export the static site: `npm run build` (make sure you have `output: ‘export’` in your `next.config.js`)
